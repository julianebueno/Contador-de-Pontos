Contador de Pontos

    Ajuda

    Este é um contador de pontos para jogos com vários jogadores.

    Para adicionar ou remover pontos de um jogador use os botões + e -.

    Os pontos são salvos no armazenamento local do navegador, então você pode fechar e abrir a página sem perder os dados.

    Para adicionar novos jogadores, clique no botão "Adicionar Jogador" e digite o nome do novo jogador.

    Para resetar os pontos de todos os jogadores, clique no botão "Resetar Pontos".

    Para remover todos os jogadores, clique no botão "Resetar Jogadores".

    Para fechar esta janela de ajuda, clique no botão "Fechar".
